## 1. Introducción
**EduBridge** es una startup del sector **EdTech** que desarrolla un ecosistema digital integrado para transformar la gestión educativa. El núcleo del proyecto es una plataforma web y móvil diseñada para centralizar el seguimiento académico, la tutoría personalizada y la gestión docente en una infraestructura basada en la nube.

La propuesta nace para resolver la desarticulación de datos en instituciones de educación superior, donde el rendimiento del alumno, la comunicación y los recursos suelen estar en sistemas separados. El objetivo principal es garantizar un aprendizaje equitativo mediante herramientas de monitoreo precisas.

---

## 2. Tecnologías Utilizadas
Para el desarrollo de la Landing Page se han empleado tecnologías modernas que garantizan un rendimiento óptimo y una estética SaaS profesional:

* **HTML5:** Estructuración semántica y accesible del contenido.
* **CSS3 (Flexbox & Grid):** Implementación de un diseño responsivo desde cero, eliminando el uso de posicionamiento absoluto para mejorar la adaptabilidad.
* **JavaScript (Vanilla):** Lógica para la interactividad del menú móvil (toggle) y navegación fluida (*smooth scroll*).
* **Boxicons:** Librería de iconos vectoriales utilizada para la interfaz de usuario y redes sociales.
* **Google Fonts (Inter):** Tipografía seleccionada para una lectura clara y moderna.

---

## 3. Características Principales del Producto
El ecosistema de **EduBridge** se destaca por:

* **Centralización de Datos:** Un solo lugar para notas, asistencia y seguimiento.
* **Tutorías Inteligentes:** Conexión directa entre alumnos y docentes con recursos multimedia.
* **Dashboard de Analítica:** Visualización en tiempo real del progreso estudiantil para detectar riesgos de deserción.
* **Diseño Adaptive/Responsive:** Experiencia de usuario optimizada para dispositivos móviles y escritorio.

## 5. Instrucciones de Ejecución
Para visualizar la Landing Page de manera local en un entorno de desarrollo:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/EduBridgee/EduBrigde-LandingPage.git](https://github.com/EduBridgee/EduBrigde-LandingPage.git)
   ```
2. **Despliegue local:**
Navegar a la carpeta raíz del proyecto e iniciar el archivo index.html en cualquier navegador web moderno.

3. **Herramientas recomendadas:**
Se sugiere el uso de la extensión Live Server en Visual Studio Code para habilitar el refresco automático (hot reload) durante la edición de los archivos de estilo.
